### Library Books Organizer

*Features:*
- CRUD books, teachers/students & personal info.
- Create logs of books system.

### Como usar | How to use:

- Start local mySQL (XAMPP recommended)
- Install .SQL script (HeidiSQL recommended)
- Open and Use.
